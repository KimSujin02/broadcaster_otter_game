# 방송하는 수달 게임
### 방송하는 수달군에게 댓글을 달아보세요!
스트리머 **수달군**에게 댓글을 달면,  
댓글의 **감정(긍정/부정)** 과 **독성 여부**를 분석해서 스트레스 수치가 변하고,  
스트레스 상태에 따라 **수달군의 표정 이미지와 반응 대사**가 달라지는  
**Streamlit 기반 인터랙티브 미니 게임**입니다.

---

## 🎮 프로젝트 소개

사용자가 댓글을 입력하면 다음 흐름으로 게임이 진행됩니다.

1. 댓글 입력
2. 감정 분석
3. 독성 분석
4. 스트레스 수치 계산
5. 스트레스 구간에 따라 수달군 상태/이미지 변경
6. 생성 모델이 댓글에 대한 짧은 반응 대사 생성
7. 최근 댓글 및 판정 로그 출력

즉, 단순 채팅형 앱이 아니라  
**댓글 반응 + 감정 변화 + 캐릭터 상태 변화**가 결합된  
작은 시뮬레이션 게임 형태의 프로젝트입니다.

---

## ✨ 주요 기능

- 댓글 감정 분석
  - 긍정 댓글이면 스트레스 감소
  - 부정 댓글이면 스트레스 증가
- 악성/독성 댓글 탐지
  - 독성 댓글은 스트레스 추가 상승
- 캐릭터 상태 변화
  - 스트레스 구간별로 다른 이미지 표시
- 캐릭터 반응 대사 생성
  - 댓글과 현재 상태를 바탕으로 한국어 한 줄 대사 생성
- 게임 로그 표시
  - 최근 댓글
  - 감정/독성 판정 결과
  - 스트레스 변화량
  - 캐릭터 반응
